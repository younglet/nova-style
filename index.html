够用就好

不做大而全，省掉动画、transform、装饰性 class。只留 IoT 控制页面真正用得到的

🎨

颜色语义化

5 个色系严格对应 IoT 状态：绿=正常，红=异常，黄=警告，蓝=信息，灰=中性

🌙

暗色模式内置

加一行 .dark 类就切换。bg-white、text-gray-900 等基础色自动适配

🧩

纯工具类

没有 .card .btn 这种黑盒组件，学生写 class 拼一拼就懂 CSS 原理

</div>
